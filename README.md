## 📁 Структура репозитория

Здесь будут размещаться материалы по каждой лабораторной работе:

```
week1/          # Лабораторная работа №1
week2/          # Лабораторная работа №2
...
```

Каждая папка содержит:
- Исходный код
- Файлы окружения (`.env`, `requirements.txt`)
- Документацию и отчёты

---

## 🚀 Быстрый старт

```bash
# Клонирование репозитория
git clone https://github.com/USERNAME/ai-course-labs.git

# Переход в папку лабораторной
cd ai-course-labs/week1

# Установка зависимостей
pip install -r requirements.txt

# Настройка переменных окружения
cp .env.example .env
# Отредактируйте .env, добавив свои API-ключи
```
